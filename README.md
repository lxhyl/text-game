# Glyph Duel

A character-art duel experiment built with `@chenglou/pretext`.

## Gameplay

- Local two-player duel
- `P1`: `WASD` to move, `F / G / H` for abilities
- `P2`: Arrow keys to move, `J / K / L` for abilities
- A full-screen three-column glyph field reflows around players and abilities in real time

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is a pure frontend Vite app and can be deployed directly to Vercel.
